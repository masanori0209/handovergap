# Acceptance Criteria

## MVP

Commands:

```bash
handovergap --help
handovergap demo
handovergap detect --scenario S001 --profile CS
handovergap evaluate
pytest
```

Must show:

- memory
- detected gaps
- clarification questions
- transferability status
- evaluation metrics

## Article-ready

Must include:

- GitHub repo
- PyPI package or TestPyPI package
- TiDB schema
- HandoverGapBench mini
- comparison results
- screenshots
- Zenn article draft

## Winning-ready

Must include:

- Streamlit demo
- PyPI release
- evaluate command with baselines
- TiDB-backed mode
- implementation pitfalls
- article-ready diagrams
