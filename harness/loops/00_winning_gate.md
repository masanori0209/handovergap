# Loop 00: Winning Gate

## Objective

Before starting or after every major loop, check whether the work improves contest-winning evidence.

## Questions

1. Does this improve the article thesis?
2. Does this produce measurable evidence?
3. Does this improve TiDB relevance?
4. Does this improve PyPI first-run UX?
5. Does this improve demo clarity?
6. Does this reduce competitor overlap?

## Stop Condition

If the answer is "no" to all, do not implement the feature.
