# Competitor Analysis

## Strong Existing Articles

### Praxia / TiDB Long-term Memory

Strengths:

- PyPI package
- GitHub
- 60s demo
- tests
- benchmark
- TiDB implementation details
- OSS feel

Risk:

- Very strong on memory store implementation.

HandoverGap Position:

> Praxia builds memory stores. HandoverGap evaluates whether stored memories are transferable to successors.

### Graphify

Strengths:

- PyPI package
- CLI UX
- knowledge graph generation
- many AI coding tool integrations
- clear product experience

HandoverGap Position:

> Graphify structures knowledge. HandoverGap evaluates whether structured/retrieved knowledge is safe to transfer to another role.

### Sayane

Strengths:

- PyPI package
- context acceptance architecture
- safe context reuse philosophy

Risk:

- Philosophically close.

HandoverGap Position:

> Sayane focuses on accepting and reusing context. HandoverGap focuses on role-conditioned missing tacit context for handover.

## Winning Angle

Do not compete on "I built a RAG app."

Compete on:

- valid-but-non-transferable memory
- role-conditioned tacit context gaps
- HandoverGapBench mini
- CLI + evaluate command
- TiDB-backed slot/evidence/gap store

## Required to Beat PyPI Competitors

- pip install experience
- CLI that works in 5 minutes
- evaluation command
- mini benchmark
- Streamlit demo
- TiDB schema and implementation
- article-ready comparison table
